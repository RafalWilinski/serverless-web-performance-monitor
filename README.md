## Install

```sh

```

## Usage

- `yarn deploy:all` compiles all lambdas, CDK Stack, Frontend application and deploys it to the cloud
- `yarn build:frontend` builds only frontend
- `yarn build:cdk` builds only CDK from TS to JS
